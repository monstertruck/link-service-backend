# BACKGROUND

- This project is the backend for a service that summarizes and categorizes links and bookmarks (URLs)
- The purpose of the service is to catalog all of the links that I might want to read
- The categories found in app/api/schemas do not have to remain static, but the ones that are currently there represent my current interests. Always suggest and request confirmation when changing categories or anything to do with the schema(s).
